# 타임로그 (TimeLog)

하루 활동을 시각적으로 기록하고 각 활동에 대한 회고를 남기는 Windows 데스크탑 위젯입니다.

## 주요 기능

- **시간 기록**: 타임라인에서 드래그하여 활동 블록을 생성 (10분 단위 스냅)
- **회고 작성**: 활동 클릭 → 잘한 점 / 아쉬운 점 / 나아갈 점 각 3개씩 기록
- **블록 편집**: 드래그로 시간대 이동, 상단/하단 엣지 드래그로 소요 시간 조절
- **날짜 이동**: 헤더 화살표로 이전/이후 날짜 확인
- **태그 검색**: 🔍 버튼 또는 트레이 메뉴에서 태그로 전체 기간 활동 검색
- **시스템 트레이**: 상주하며 언제든 위젯 표시/숨김 전환

## 설치 및 실행

### 요구 사항

- Python 3.10 이상
- PyQt6

### 설치

```bash
pip install PyQt6
```

### 실행

```bash
python main.py
```

## 사용법

| 동작 | 기능 |
|---|---|
| 타임라인 드래그 | 새 활동 추가 |
| 활동 블록 클릭 | 회고 작성 / 수정 |
| 활동 블록 드래그 | 시간대 이동 |
| 상단/하단 엣지 드래그 | 소요 시간 조절 |
| 활동 블록 우클릭 | 삭제 메뉴 |
| 헤더 ◀ / ▶ | 날짜 이동 |
| 헤더 드래그 | 위젯 위치 이동 |
| 🔍 버튼 | 태그 검색 창 열기 |

## 데이터 저장

모든 데이터는 `timelog.db` (SQLite) 파일에 로컬 저장됩니다.

## 파일 구조

```
├── main.py        진입점, 시스템 트레이
├── widget.py      메인 위젯 창 (헤더 + 스크롤 영역)
├── timeline.py    타임라인 UI, 드래그/클릭 로직
├── dialogs.py     활동 추가 / 회고 작성 팝업
├── search.py      태그 검색 창
├── database.py    SQLite CRUD
├── models.py      Activity, Reflection 데이터 모델
└── timelog.db     데이터 파일 (실행 후 자동 생성)
```
